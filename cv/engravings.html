이미지 각인(인그레이빙) 예시

컴퓨터 비전과 이미지 처리 기술을 활용한 현실적인 텍스트 효과

각인 예시 갤러리

문패 인그레이빙

대리석 문패에 자연스러운 각인 효과를 적용한 예시입니다. 3D 조명 효과와 노이즈를 통해 실제 조각과 유사한 질감을 구현했습니다.

세로쓰기 3D 각인 조명 효과 자연스러운 질감
도자기 곡면 텍스트

주전자 곡면에 금색 텍스트를 자연스럽게 매핑한 예시입니다. 원근 변환과 조명 효과를 통해 곡면에 맞는 현실적인 텍스트 효과를 구현했습니다.

곡면 매핑 금색 효과 반사광 원근 변환

사용된 기술

문패 인그레이빙 기술
  • 세로쓰기 텍스트 렌더링
  • 가우시안 블러 마스크 처리
  • Sobel 필터 그라디언트 계산
  • 3D 법선 벡터 계산
  • 조명 벡터 내적 효과
  • 노이즈 추가 질감 생성
  • 원형 모서리 마스크
도자기 곡면 매핑 기술
  • HoughCircles 원형 검출
  • Contour 기반 타원 fitting
  • 원근 변환 행렬 계산
  • Tight bounding box 이미지
  • 금색 반사 효과 렌더링
  • 알파 채널 분리/합성
  • 곡면 투영 warping
문패 인그레이빙 상세보기
기술적 특징:

• 3D 조명 효과로 깊이감 있는 각인
• 세로쓰기 레이아웃으로 전통적 느낌
• 노이즈와 블러로 자연스러운 질감
• 원형 모서리로 세련된 디자인

도자기 곡면 텍스트 상세보기
기술적 특징:

• 원근 변환으로 곡면에 자연스럽게 매핑
• 금색 반사광과 하이라이트 효과
• Tight bounding box로 아티팩트 방지
• 조명 벡터 기반 현실적 렌더링
